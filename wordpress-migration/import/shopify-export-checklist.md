# Shopify Export Checklist

Needed before the real import can be written and run.

- Products CSV export.
- Product variants with IDs, SKUs, prices, inventory, and options.
- Product media export or CDN URL list.
- Product metafields:
  - `spec.cas`
  - `spec.purity`
  - `details.cas`
  - `details.formula`
  - `details.molecular_weight`
  - `details.sequence`
  - `details.form`
  - `details.solubility`
  - `details.storage`
- COA metaobjects:
  - `coa.batches`
  - `coa.batch`
  - all referenced `coa_batch` fields
- Shopify files used by COA records.
- Pages.
- Blog posts/articles.
- Navigation menus.
- Redirects.
- Theme settings data for global values.
- App list with role/owner.
